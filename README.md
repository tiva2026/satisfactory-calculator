# Satisfactory Production Calculator

A mobile-first production calculator for [Satisfactory](https://www.satisfactorygame.com/). Enter any item and target rate, and the calculator will work out the full production chain for you.

Live site: [https://satis.cc](https://satis.cc)

## What it does

- Calculates the full production tree for any item
- Breaks down intermediate ingredients and raw resource needs
- Shows how many buildings are required for each step
- Displays the total power consumption
- Works well on mobile and desktop
- Uses official Satisfactory item and recipe data

## How to use

1. Search for an item
2. Enter your target rate
3. View the production plan and resource breakdown
